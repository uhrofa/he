Getting started

HTMLEdito, meet the nice and simple HTML, CSS and Javascript editor. Just Run index.html file with your modern browser. No need installation and you are ready to go.

HTMLEdito is a client-side app, no need server. The only one thing you should be aware is NEVER REFRESH or CLOSE TAB/WINDOW Browser when you are working with final code. Always save to another place (native code editor). If you refresh the browser, it would reload the default code. So, all your codes that have been written would be lost.

Run (JS)

Run (JS) button is used for Run javascript code manually. This button is working if you disable (Uncheck) Auto Run JS in menu Setting.

Library

Add custom library with just a click button. This is very useful when you are working with popular HTML, CSS and Javascript Library like bootstrap, uikit, Material Design Lite, jQuery, jQueryUI and so on. You can add custom library by editing library.js in assets folder and index.html for custom menu.

Settings

This settings is optional, give it a try to meet your need. The available settings are as follow:

  • Auto Run JS. If you enable this, all your javascript codes in javascript panel would be run automatically and vice versa. Enabled by default.
  • Word Wrap. Wrap the codes to a newline in all panels you are working with. Disabled by default.
  • Auto Close Tag. Write the first tag, and the closing tag would be created for you automatically.
  • Highlight Layer Mode. Layer mode in the preview panel would be highlighted.

Preview Size

Work with any size preview window. Add your custom width and height in the input box. This is useful to test your code through mobile viewport.

Emmet

HTMLEdito supports emmet. Please, refer to Emmet Documentation. You can use Default keybindings:

  • Cmd-E or Tab: Expand abbreviation
  • Cmd-D: Balance Tag (matches opening and closing tag pair)
  • Shift-Cmd-D: Balance Tag Inward
  • Shift-Cmd-A: Wrap With Abbreviation
  • Ctrl-Alt-Right: Next Edit Point
  • Ctrl-Alt-Left: Previous Edit Point
  • Cmd-L: Select line
  • Cmd-Shift-M: Merge Lines
  • Cmd-/: Toggle Comment
  • Cmd-J: Split/Join Tag
  • Cmd-K: Remove Tag
  • Shift-Cmd-Y: Evaluate Math Expression
  • Ctrl-Up: Increment Number by 1
  • Ctrl-Down: Decrement Number by 1
  • Ctrl-Alt-Up: Increment Number by 0.1
  • Ctrl-Alt-Down: Decrement Number by 0.1
  • Shift-Ctrl-Up: Increment Number by 10
  • Shift-Ctrl-Down: Decrement Number by 10
  • Shift-Cmd-.: Select Next Item
  • Shift-Cmd-,: Select Previous Item
  • Cmd-B: Reflect CSS Value
